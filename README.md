## Get started
```bash
#1. Open your terminal and clone the repository
git clone git@github.com:Yu19980809/project-notion-clone.git

#2. Enter the project you just cloned
cd project-notion-clone

#3. Download all dependencies
npm install

#4. Copy the .env.example variables into a separate .env file
#5. Fill all info listed in .env

#6. Run the project
npm run dev
```
